# Customer Segmentation using K-Means

## Overview
This project performs clustering on customer data to segment users based on purchasing behavior.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## Features
- Data preprocessing and feature scaling
- K-Means clustering
- Visualization of customer segments

## Results
- Identified distinct customer groups
- Enabled personalized marketing strategies
